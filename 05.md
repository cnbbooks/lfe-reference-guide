# Older Scheme-inspired macros

```scheme
(define (name arg ...) ...)
```

```scheme
(define name lambda|match-lambda)
```

```scheme
(define-syntax name
  (syntax-rules (pat exp) ...)|(macro (pat body) ...))
```

```scheme
(let-syntax ((name ...)
             ...)
  ...)
```

```scheme
(begin ...)
```

```scheme
(define-record name ...)
```
