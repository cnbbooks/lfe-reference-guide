# Special syntactic rules
| ``#b``, ``#o``, ``#d``, ``#x``, and  ``#23r`` | Based integers |
| -- | -- |
| ``#(e e ... )`` | Tuple constants |
| ``#b(e e ... )`` | Binary constants, ``e`` ... are valid literals segments |
| ``#m(k v ... )`` | Map constants, ``k`` ``v`` are keys and values |
| ``[ ... ]`` | Allowed as alternative to ``( ... )`` |
| 0:6 | 1:6 |
